# Next.js eCommerce / Product Management App

## Project Description
This is a simple eCommerce-style product management application built using **Next.js (App Router)** with protected and public pages. Users can view products, add new products, edit existing products, and delete products. Authentication is handled via **Firebase** (or NextAuth.js if you choose).  

The app focuses on:
- Polished and responsive UI
- Consistent layout with Tailwind CSS
- Protected pages for authenticated users
- Clean and reusable components



## Features
- Public Pages:
  - Landing/Home page with hero, feature sections, and product cards
  - About page
  - Contact page
  - Products List page with search and "View Details"
  - Product Detail page
- Protected Pages (Require Login):
  - Add Product page
  - Manage Products page (Edit, Delete, View)
  - Edit Product page



## Setup & Installation

- Node.js 
- npm 
- MongoDB / Express.js backend
- Firebase project
